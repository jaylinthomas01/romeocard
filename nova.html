Nova

Status: Actively in development

Overview: Nova is a mood-aware AI voice assistant designed to flirt, sass, and help you get shit done. She’s locally hosted, customizable, and built on ESPHome, Raspberry Pi, and custom TTS models.

Tech Stack

  • ESP32 (Home Assistant integration)
  • ElevenLabs / XTTS2 for voice
  • Python + shell scripts for actions
  • Local TTS + optional AI memory module

What’s Next

  • Add voice triggers (wake word)
  • Integrate with Google Home + Spotify
  • Build GUI dashboard for toggles
